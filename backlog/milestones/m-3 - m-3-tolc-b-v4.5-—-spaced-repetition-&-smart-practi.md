## Description

Introduce spaced repetition scheduling and intelligent practice sessions to optimize learning efficiency. Core SRS engine with lightweight per-type×difficulty tracking, smart daily interleaved sessions, wrong-answer reconsolidation, and problem variation. This milestone covers Phase 1 of the adaptive learning system — the scheduling and retrieval practice layer — while Phase 2 (adaptive difficulty + time management) remains in m-2.
